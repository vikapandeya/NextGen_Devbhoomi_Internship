Beginner Level — 1 Month (4 Weeks)

SQL Fundamentals · MySQL · Joins · MongoDB Basics · ER Diagrams · Schema Design

Beginner
Learning Objectives
  • Write SQL queries for all CRUD operations
  • Design normalized relational database schemas (1NF, 2NF, 3NF)
  • Use JOINs and subqueries to answer complex business questions
  • Perform CRUD operations in MongoDB with the aggregation pipeline
  • Create ER diagrams for real-world entities
  • Administer a MySQL instance (users, permissions, backups)

Sample Viva Questions

Beginner Viva

  • What is the difference between DELETE, TRUNCATE, and DROP?
  • Explain 1NF, 2NF, and 3NF with an example.
  • When would you use a LEFT JOIN vs INNER JOIN?
  • What is a primary key vs a foreign key?

Intermediate Viva

  • Explain the difference between clustered and non-clustered indexes.
  • What are ACID properties? Give an example of each.
  • What is the purpose of the VACUUM command in PostgreSQL?
  • How does Redis handle cache eviction when memory is full?

Advanced Viva

  • How do you choose a shard key for a MongoDB sharded cluster?
  • What is WAL (Write-Ahead Log) and why is it important for recovery?
  • How would you handle a production database with 0 downtime during a major schema change?
  • Explain the difference between RTO and RPO and how you design for them.
