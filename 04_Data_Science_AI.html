Beginner Level — 1 Month (4 Weeks)

Python · NumPy · Pandas · Matplotlib · EDA · ML Basics · Streamlit Deployment

Beginner
Learning Objectives
  • Write Python scripts for data manipulation and analysis
  • Perform EDA (Exploratory Data Analysis) on real datasets
  • Build and evaluate basic supervised ML models
  • Create interactive data visualizations
  • Deploy an ML model as a Streamlit web app
  • Submit a Kaggle competition entry

Sample Viva Questions

Beginner Viva

  • What is the difference between supervised and unsupervised learning?
  • Explain precision vs recall. When do you prefer each?
  • What is overfitting and how do you prevent it?
  • What does PCA do to a dataset?

Intermediate Viva

  • Why does XGBoost outperform Random Forest in most tabular tasks?
  • Explain the vanishing gradient problem and how LSTMs solve it.
  • What is the attention mechanism in Transformers?
  • What is RAG and why is it better than pure fine-tuning for knowledge?

Advanced Viva

  • Explain LoRA fine-tuning and how it reduces memory requirements.
  • How do you evaluate RAG pipeline quality (relevance + faithfulness)?
  • What is model drift and how do you detect it in production?
  • Design a multi-agent system for automated code review.
