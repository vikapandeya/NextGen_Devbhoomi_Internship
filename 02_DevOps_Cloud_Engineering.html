Beginner Level — 1 Month (4 Weeks)

Linux Administration · Networking · AWS Basics · Docker Introduction

Beginner
Learning Objectives
  • Navigate and manage a Linux system (files, users, permissions)
  • Understand OSI model, TCP/IP, subnetting, and DNS
  • Launch and configure AWS EC2 instances with VPC/security groups
  • Set up S3 static website and use AWS CLI
  • Containerize and run an application with Docker
  • Write basic shell scripts for automation
Week-by-Week Curriculum
WeekModuleTopics CoveredHands-on LabAssignment
Week 1 Linux Administration Filesystem hierarchy, ls/cd/cp/mv/rm, users & permissions (chmod/chown), package management (apt/yum), process management (ps/top/kill/systemd), shell scripting basics Install LAMP stack, configure Apache, create multi-user environment Write a disk usage monitoring script with email alert simulation
Week 2 Networking Fundamentals OSI model, TCP/IP, IP addressing, subnetting, DNS/DHCP, SSH key-based auth, UFW firewall, curl/wget/nmap/netstat, VPN concepts Configure SSH passwordless auth, UFW rules (ports 22/80/443 only) Subnet a /24 into 4 subnets + audit open ports on localhost
Week 3 AWS Cloud Fundamentals AWS global infra, IAM (users/roles/policies/MFA), EC2 launch & key pairs, Security Groups, S3 buckets & policies, VPC + subnets + IGW, AWS CLI Launch Ubuntu EC2, deploy Apache; create custom VPC with public/private subnets Host static HTML on EC2 + S3 with versioning + billing alert setup
Week 4 Docker Introduction Docker concepts (images/containers/layers), Dockerfile, Docker Compose, Docker Hub, image tagging, multi-container apps Dockerize a Node.js app + Docker Compose with app + MySQL Deploy Dockerized app to EC2 + push image to Docker Hub

Sample Viva Questions

Beginner Viva

  • What is the difference between chmod 755 and 644?
  • Explain how DHCP assigns IP addresses.
  • What is an IAM role vs IAM user in AWS?
  • How does a Dockerfile build an image layer by layer?

Intermediate Viva

  • What is the difference between a Deployment and a StatefulSet?
  • How does HPA decide to scale pods?
  • Explain Terraform state and why remote state matters.
  • What is the purpose of Ansible idempotency?

Advanced Viva

  • How does Istio implement mTLS between services?
  • Explain the difference between RTO and RPO.
  • How would you design a zero-downtime deployment strategy?
  • What is an error budget and how do you calculate it?
