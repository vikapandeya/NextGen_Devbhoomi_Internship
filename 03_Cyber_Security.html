⚠️ Ethical Use Declaration

All security testing tools, techniques, and methodologies taught in this program are strictly for authorized, controlled lab environments, CTF (Capture The Flag) competitions, and defensive security purposes only. Any unauthorized use of these techniques against systems without explicit written permission is illegal and strictly prohibited. Students must sign an Ethical Use Agreement before beginning this domain.

Beginner Level — 1 Month (4 Weeks)

Security Fundamentals · Network Security · OWASP Top 10 · Basic Scanning

Beginner
Learning Objectives
  • Understand CIA Triad and core cybersecurity principles
  • Identify and classify common attack types and threat actors
  • Perform basic network reconnaissance with Nmap and Wireshark
  • Understand OWASP Top 10 vulnerabilities with DVWA
  • Use Burp Suite for basic web interception and testing
  • Write a structured security audit/vulnerability report
Weekly Curriculum
WeekTopicsLabAssignment
Week 1 CIA Triad (Confidentiality, Integrity, Availability), attack types (phishing, MITM, DoS, social engineering), encryption basics (AES, RSA, SHA), public/private key infrastructure Analyze phishing email samples; encrypt/decrypt using OpenSSL Map a realistic social engineering attack scenario with countermeasures
Week 2 OSI security layers, TLS/HTTPS deep dive, SSH, VPN types (SSL/IPSec), Nmap (host discovery, port/service/OS scanning), Wireshark (packet capture and protocol analysis) Capture and compare HTTP vs HTTPS traffic in Wireshark; Nmap scan lab VM Full Nmap scan report of a target lab machine with open ports and services
Week 3 OWASP Top 10 (SQLi, XSS, CSRF, IDOR, broken auth, security misconfiguration), DVWA setup and exploitation (Low security level), Burp Suite: intercept, repeater, decoder Exploit SQLi and Stored XSS on DVWA; intercept login request with Burp Document 3 OWASP vulnerabilities found with impact, evidence, and remediation
Week 4 OpenVAS/Nessus basic scan, vulnerability scoring (CVSS v3), security report writing, TryHackMe beginner rooms (Pre-Security path) Run OpenVAS scan on lab VM; complete 3 TryHackMe beginner rooms Professional security audit report for a lab web application

Intermediate Level — 3 Months (12 Weeks)

Network Pentesting · Web App Pentesting · API Security · SOC Operations

Intermediate
Learning Objectives
  • Conduct structured penetration tests following PTES/OWASP methodology
  • Exploit network vulnerabilities in controlled lab environments
  • Perform advanced web app testing (SQLi, XSS, SSRF, JWT, API security)
  • Monitor and respond to security incidents using SIEM (Wazuh)
  • Assess AWS security misconfigurations
  • Write professional penetration test reports with CVSS scoring

Sample Viva Questions

Beginner Viva

  • What is the CIA Triad and why does it matter?
  • Explain the difference between a vulnerability and an exploit.
  • What does Nmap's -sV flag do?
  • Name 3 items from OWASP Top 10 and explain one.

Intermediate Viva

  • Explain how SQL injection works with an example.
  • What is the difference between reflected and stored XSS?
  • How does JWT token forgery with 'none' algorithm work?
  • What is the kill chain and how does it relate to pentesting?

Advanced Viva

  • Explain Kerberoasting and how to detect it.
  • What is lateral movement in the MITRE ATT&CK framework?
  • How would you design a SOC monitoring strategy for a 500-employee company?
  • What is the difference between DAST and SAST?
