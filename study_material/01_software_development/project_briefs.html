Project Briefs

Three capstone projects — one per level. Each project is submitted for evaluation. Your best project becomes the centerpiece of your portfolio.

3 Projects Beginner · Intermediate · Advanced Portfolio Projects
Project 1 — Beginner Level
P1

Personal Portfolio Website

HTML5 · CSS3 · JavaScript · GitHub Pages

Beginner

Project Description

Build a professional personal portfolio website that showcases who you are, your skills, and your projects. This is the page you will share with employers and use throughout the internship program.

Requirements

  • Multi-section single page: Hero, About, Skills, Projects, Contact
  • Sticky navigation with smooth scroll to sections
  • Skills displayed in a responsive CSS Grid layout
  • Projects section showing at least 2 projects with description, tech stack, and links
  • Contact form with JavaScript validation (name, email, message — all required)
  • Fully responsive: tested and working on mobile (360px) and desktop (1280px)
  • Deployed live on GitHub Pages
  • Zero HTML validation errors at validator.w3.org

Bonus (Optional)

  • Dark mode toggle using CSS variables and JavaScript
  • Smooth CSS animations on scroll (Intersection Observer API)
  • Custom favicon and Open Graph meta tags

Project 2 — Intermediate Level
P2

Full-Stack Expense Tracker

React · Node.js · Express · MySQL · Vercel + Render

Intermediate

Project Description

Build a full-stack expense tracking application where users can add, categorize, view, and delete expense entries. Data persists in a real MySQL database via a REST API.

Requirements

  • React frontend with components for: ExpenseForm, ExpenseList, ExpenseItem, Summary
  • Add expense: amount, description, category (food/travel/shopping/other), date
  • View all expenses sorted by date (newest first)
  • Delete any expense
  • Summary: total spent, total by category
  • Filter expenses by category
  • Express REST API: GET, POST, DELETE /api/expenses
  • MySQL table: id, amount, description, category, date, created_at
  • Environment variables for all DB credentials
  • CORS configured for your frontend domain only
  • Both frontend and backend deployed live

Bonus (Optional)

  • Chart showing spending by category (Chart.js or CSS bars)
  • Monthly view — filter by month/year
  • Export expenses as CSV

Project 3 — Advanced Level
P3

Production-Ready Blog Platform

TypeScript · React · Node.js · MySQL · Jest · GitHub Actions

Advanced

Project Description

Build a complete blogging platform with user authentication, full CRUD for posts, and comments. The codebase must be TypeScript throughout, have ≥70% test coverage, and a working CI/CD pipeline.

Requirements

  • TypeScript on both frontend (React) and backend (Node/Express)
  • User registration and login (JWT authentication)
  • Create, read, update, delete blog posts
  • Comments on posts (add + delete own comments)
  • Posts list page + individual post page
  • Rich text editor for writing posts (use a simple one like Quill.js)
  • Image upload for post cover image
  • Unit tests with Jest: ≥70% statement coverage
  • Component tests with React Testing Library
  • GitHub Actions CI pipeline: type-check + test + build on every push
  • Branch protection: main requires CI to pass
  • Dockerfile for the backend API
  • Lighthouse score ≥ 80 on Performance
