Intermediate Lab Guide

4 labs building a full-stack Task Manager application. React frontend + Express backend + MySQL database — connected together and deployed live.

4 Labs (Weeks 5–8) React · Node.js · MySQL · Postman Intermediate Level

Lab Setup — Prerequisites

  • Node.js 20+ installed: download from nodejs.org
  • XAMPP installed (includes MySQL): apachefriends.org — start MySQL in XAMPP Control Panel
  • Postman installed: postman.com/downloads
  • Verify Node: open terminal and run node --version (should show v20.x.x)
Lab 5 — React Task Manager UI (Week 5)
1

Create Vite + React project

npm create vite@latest task-manager -- --template react
cd task-manager
npm install
npm run dev

Open http://localhost:5173 — you should see the Vite + React welcome page.

2

Create TaskItem component

Create src/components/TaskItem.jsx:

function TaskItem({ task, onToggle, onDelete }) {
  return (
    <div className={`task-item ${task.done ? 'done' : ''}`}>
      <span className="task-title">{task.title}</span>
      <div className="task-actions">
        <button onClick={() => onToggle(task.id)}>
          {task.done ? 'Undo' : 'Done'}
        </button>
        <button onClick={() => onDelete(task.id)}>Delete</button>
      </div>
    </div>
  );
}

export default TaskItem;
3

Build App.jsx with state management

import { useState } from 'react';
import TaskItem from './components/TaskItem';

const INITIAL_TASKS = [
  { id: 1, title: 'Learn React hooks', done: false },
  { id: 2, title: 'Build task manager', done: false },
  { id: 3, title: 'Deploy to Vercel', done: false },
];

function App() {
  const [tasks, setTasks]   = useState(INITIAL_TASKS);
  const [newTask, setNewTask] = useState('');

  const addTask = () => {
    const title = newTask.trim();
    if (!title) return;
    setTasks([...tasks, { id: Date.now(), title, done: false }]);
    setNewTask('');
  };

  const toggleTask = (id) =>
    setTasks(tasks.map(t => t.id === id ? { ...t, done: !t.done } : t));

  const deleteTask = (id) =>
    setTasks(tasks.filter(t => t.id !== id));

  return (
    <div className="app">
      <h1>Task Manager</h1>
      <div className="add-task">
        <input
          value={newTask}
          onChange={(e) => setNewTask(e.target.value)}
          onKeyDown={(e) => e.key === 'Enter' && addTask()}
          placeholder="Add a new task..."
        />
        <button onClick={addTask}>Add</button>
      </div>
      <div className="task-list">
        {tasks.map(t =>
          <TaskItem key={t.id} task={t} onToggle={toggleTask} onDelete={deleteTask} />
        )}
      </div>
      <p className="counter">{tasks.filter(t => !t.done).length} remaining</p>
    </div>
  );
}

export default App;
4

Add CSS and test all interactions

Style your App in src/App.css. Test: add a task, complete a task, delete a task, press Enter to add. All must work. Take a screenshot showing the task list with mixed completed/pending tasks.

Lab 6 — Express REST API (Week 6)
1

Create Express server

mkdir task-api
cd task-api
npm init -y
npm install express cors

Create index.js:

const express = require('express');
const cors    = require('cors');
const app     = express();

app.use(cors());
app.use(express.json());

let tasks = [
  { id: 1, title: 'Build API', done: false },
];
let nextId = 2;

app.get('/api/tasks',        (req, res) => res.json(tasks));

app.post('/api/tasks', (req, res) => {
  const { title } = req.body;
  if (!title) return res.status(400).json({ error: 'title required' });
  const task = { id: nextId++, title, done: false };
  tasks.push(task);
  res.status(201).json(task);
});

app.patch('/api/tasks/:id', (req, res) => {
  const task = tasks.find(t => t.id === +req.params.id);
  if (!task) return res.status(404).json({ error: 'not found' });
  task.done = !task.done;
  res.json(task);
});

app.delete('/api/tasks/:id', (req, res) => {
  tasks = tasks.filter(t => t.id !== +req.params.id);
  res.status(204).end();
});

app.listen(3001, () => console.log('API running on port 3001'));
2

Test every endpoint in Postman

  • GET http://localhost:3001/api/tasks → returns array
  • POST http://localhost:3001/api/tasks with body {"title":"New task"} → returns new task with id
  • PATCH http://localhost:3001/api/tasks/1 → toggles done
  • DELETE http://localhost:3001/api/tasks/1 → 204 no content
  • GET again → deleted task is gone

Screenshot each test result in Postman.

Lab 7 — Connect API to MySQL Database (Week 7)
1

Create the database

Open XAMPP → Start MySQL → open phpMyAdmin (localhost/phpmyadmin). Create a new database called taskdb. Run this SQL:

CREATE TABLE tasks (
  id    INT AUTO_INCREMENT PRIMARY KEY,
  title VARCHAR(255) NOT NULL,
  done  TINYINT(1)   DEFAULT 0
);
2

Install MySQL driver and dotenv

npm install mysql2 dotenv

Create .env file in task-api folder:

DB_HOST=localhost
DB_USER=root
DB_PASS=
DB_NAME=taskdb

Create .gitignore file and add: .env (one line)

3

Update index.js to use real database

require('dotenv').config();
const mysql = require('mysql2/promise');

const db = mysql.createPool({
  host:     process.env.DB_HOST,
  user:     process.env.DB_USER,
  password: process.env.DB_PASS,
  database: process.env.DB_NAME,
});

app.get('/api/tasks', async (req, res) => {
  try {
    const [rows] = await db.query('SELECT * FROM tasks ORDER BY id DESC');
    res.json(rows);
  } catch (err) { res.status(500).json({ error: 'DB error' }); }
});

app.post('/api/tasks', async (req, res) => {
  const { title } = req.body;
  if (!title) return res.status(400).json({ error: 'title required' });
  try {
    const [result] = await db.query(
      'INSERT INTO tasks (title) VALUES (?)', [title]
    );
    res.status(201).json({ id: result.insertId, title, done: false });
  } catch (err) { res.status(500).json({ error: 'DB error' }); }
});
4

Verify in Postman + phpMyAdmin

POST a new task via Postman, then refresh phpMyAdmin → taskdb → tasks table. The row must appear. DELETE a task via Postman, verify it's removed from the table. Screenshot both.

Lab 8 — Connect React to the Real API + Deploy (Week 8)
1

Update React to fetch from Express

import { useState, useEffect } from 'react';

const API = import.meta.env.VITE_API_URL || 'http://localhost:3001';

function App() {
  const [tasks, setTasks]   = useState([]);
  const [loading, setLoading] = useState(true);
  const [newTask, setNewTask] = useState('');

  useEffect(() => {
    fetch(`${API}/api/tasks`)
      .then(r => r.json())
      .then(data => { setTasks(data); setLoading(false); });
  }, []);

  const addTask = async () => {
    const title = newTask.trim();
    if (!title) return;
    const res  = await fetch(`${API}/api/tasks`, {
      method: 'POST',
      headers: { 'Content-Type': 'application/json' },
      body: JSON.stringify({ title }),
    });
    const task = await res.json();
    setTasks([task, ...tasks]);
    setNewTask('');
  };

  if (loading) return <p>Loading tasks...</p>;

  return (
    <div className="app">
      <h1>Task Manager</h1>
      <div className="add-task">
        <input value={newTask}
          onChange={e => setNewTask(e.target.value)}
          onKeyDown={e => e.key === 'Enter' && addTask()}
          placeholder="New task..." />
        <button onClick={addTask}>Add</button>
      </div>
      {tasks.map(t => <div key={t.id}>{t.title}</div>)}
    </div>
  );
}
2

Deploy backend to Render

Push task-api to a new GitHub repo. Go to render.com → New Web Service → Connect your GitHub repo. Set:

  • Build Command: npm install
  • Start Command: node index.js
  • Add environment variables from your .env file
3

Deploy frontend to Vercel

Push task-manager to GitHub. Import at vercel.com. Add environment variable: VITE_API_URL = https://your-api.onrender.com. Deploy. Test the live app — add a task, it should persist in the database.
