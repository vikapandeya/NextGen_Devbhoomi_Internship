Lab 1

Setting Up Your Data Science Environment

Week 1 45 min
Objective: Get your Python data science environment running, install the core libraries, and make your first contact with real data.
Step 1 Choose Your Environment

Option A — Google Colab (Recommended for beginners, no installation)

  1. Open your browser and go to colab.research.google.com
  2. Sign in with your Google account
  3. Click New Notebook — this creates a .ipynb file in your Google Drive
  4. All major DS libraries (pandas, NumPy, matplotlib, seaborn, scikit-learn) are pre-installed
  5. Click on a code cell and press Ctrl+Enter to run it

Option B — Anaconda (Local installation)

  1. Go to anaconda.com/download and download the installer for your OS
  2. Run the installer with default settings
  3. Open Anaconda Navigator → Launch Jupyter Notebook
  4. Or open a terminal and type: jupyter notebook

Lab 3

Data Cleaning Pipeline

Week 3 60 min
Objective: Build a complete data cleaning pipeline that transforms the raw Titanic dataset into a clean, analysis-ready CSV file.
Step 1 Load Raw Data and Assess Quality
>import pandas as pd
import seaborn as sns

df = sns.load_dataset('titanic')
print("RAW DATA QUALITY REPORT")
print("="*40)
print(f"Rows:    {df.shape[0]}")
print(f"Columns: {df.shape[1]}")
print(f"\nDuplicate rows: {df.duplicated().sum()}")
print(f"\nMissing values:")
missing_pct = (df.isnull().sum() / len(df) * 100).round(1)
print(missing_pct[missing_pct > 0])
Step 2 Drop High-Missing Columns
>df_clean = df.copy()

# Drop columns with more than 50% missing values
threshold = len(df_clean) * 0.5
cols_before = df_clean.shape[1]
df_clean = df_clean.dropna(thresh=threshold, axis=1)
cols_after = df_clean.shape[1]
print(f"Dropped {cols_before - cols_after} columns with >50% missing")
print(f"Remaining columns: {df_clean.columns.tolist()}")

# Also drop columns that won't be useful for analysis
cols_to_drop = ['deck', 'embark_town', 'alive', 'who', 'adult_male', 'alone', 'class']
cols_to_drop = [c for c in cols_to_drop if c in df_clean.columns]
df_clean = df_clean.drop(columns=cols_to_drop)
print(f"\nAfter dropping utility columns: {df_clean.columns.tolist()}")
Step 3 Fill Missing Values
># Fill missing 'age' with the median
median_age = df_clean['age'].median()
df_clean['age'] = df_clean['age'].fillna(median_age)
print(f"Filled missing 'age' with median: {median_age:.1f} years")

# Fill missing 'embarked' with the mode
mode_embarked = df_clean['embarked'].mode()[0]
df_clean['embarked'] = df_clean['embarked'].fillna(mode_embarked)
print(f"Filled missing 'embarked' with mode: '{mode_embarked}'")

# Fill missing 'fare' with the median by class
for pclass in [1, 2, 3]:
    median_fare = df_clean[df_clean['pclass'] == pclass]['fare'].median()
    mask = (df_clean['pclass'] == pclass) & (df_clean['fare'].isnull())
    df_clean.loc[mask, 'fare'] = median_fare

# Verify no more missing values
print("\nMissing values after cleaning:")
print(df_clean.isnull().sum())
Step 4 Remove Duplicates & Encode Categories
># Remove any duplicate rows
before = len(df_clean)
df_clean = df_clean.drop_duplicates()
print(f"Removed {before - len(df_clean)} duplicate rows")

# Encode 'sex' as binary (male=1, female=0)
df_clean['sex_enc'] = df_clean['sex'].map({'male': 1, 'female': 0})
print(f"\nSex encoding: {df_clean[['sex','sex_enc']].drop_duplicates()}")

# Encode 'embarked' with one-hot encoding
df_clean = pd.get_dummies(df_clean, columns=['embarked'], prefix='emb', drop_first=True, dtype=int)

# Convert pclass to integer
df_clean['pclass'] = df_clean['pclass'].astype(int)

print("\nFinal dtypes:")
print(df_clean.dtypes)
Step 5 Export Cleaned Dataset
># Preview final clean dataset
print("CLEAN DATASET PREVIEW:")
print(df_clean.head())
print(f"\nShape: {df_clean.shape}")
print(f"No missing values: {df_clean.isnull().sum().sum() == 0}")

# Save to CSV
df_clean.to_csv('titanic_clean.csv', index=False)
print("\nSaved: titanic_clean.csv")

# Verify by reloading
df_verify = pd.read_csv('titanic_clean.csv')
print(f"Verified — Reloaded shape: {df_verify.shape}")

># Iris Dataset — Exploratory Data Analysis
## NextGen Devbhoomi Foundation — Batch 2026–27

This notebook performs a complete exploratory data analysis of the famous Iris flower dataset,
first published by statistician Ronald Fisher in 1936. The dataset contains measurements of
150 iris flowers from 3 species: Setosa, Versicolor, and Virginica.

**Key Questions:**
1. Are the four measurements (sepal/petal length/width) different across species?
2. Which measurements best distinguish one species from another?
3. Are there any strong correlations between features?
>from sklearn.datasets import load_iris
import pandas as pd
import numpy as np
import matplotlib.pyplot as plt
import seaborn as sns

# Load dataset
iris = load_iris(as_frame=True)
df = iris.frame

# Map species codes to names
df['species'] = df['target'].map({0: 'setosa', 1: 'versicolor', 2: 'virginica'})
df = df.drop('target', axis=1)

print("Dataset loaded successfully!")
print(f"Shape: {df.shape}")
print(f"Species: {df['species'].unique()}")

Section 5 Conclusions & Export

Add this as a Markdown cell:

>## Key Findings — Iris EDA

1. **Petal dimensions are the strongest separators**: Setosa has much smaller petal length
   (mean 1.46 cm) vs Versicolor (4.26 cm) and Virginica (5.55 cm).

2. **Petal length and petal width are highly correlated** (r = 0.96) — plants with longer
   petals also tend to have wider petals.

3. **Setosa is easily separable** from the other two species in all feature combinations.
   Versicolor and Virginica overlap in sepal dimensions but are separable by petal size.

4. **Sepal width is the least discriminatory feature** — all three species have similar
   sepal width distributions with significant overlap.

5. **No missing values** — This is an unusually clean dataset, making it ideal for
   learning ML algorithms.
