3Programme Skill Milestones
🌱
Foundational — Weeks 1–4
Run an Expo app on a real device, core components and layout, basic navigation between screens, persist data with AsyncStorage, submit the Habit Tracker project.
⚡
Application — Weeks 5–8
Integrate a real REST API, handle loading and error states, use device features (camera, location), offline-first data with SQLite, submit the Recipe Finder project.
🚀
Professional — Weeks 9–12
Reanimated animations, full test suite with ≥60% coverage, EAS CI/CD pipeline, APK installable on device, published to Google Play internal testing.
💼
Job-Ready Checklist
2 apps on Google Play or TestFlight, GitHub profile with mobile repos, APK screenshots and demo video on LinkedIn, 1 certification completed, 2 mock interviews done.
490-Day Post-Programme Action Plan

Phase 1 — Days 1–30: Polish & Publish

Fix known bugs across your projects, add loading and error states everywhere, write a clear README for each app repository, publish APKs to Google Play internal testing, and record a 60-second demo video for each app to post on LinkedIn.

Phase 2 — Days 31–60: Apply & Sharpen

Apply to React Native roles on Naukri, LinkedIn, and AngelList India (target 5 applications per week), solve logic puzzles in JavaScript (Advent of Code is excellent), join a React Native Discord or Slack community, and attempt one Expo tutorial project outside your comfort zone.

Phase 3 — Days 61–90: Interview & Close

Prepare the top 20 React Native interview questions (explain the JS bridge, FlatList optimization, AsyncStorage vs SQLite, state management choices), research each company's mobile stack before every interview, and follow up with every recruiter within 24 hours of any meeting.
