Lab 1 — Hello Expo (Week 1)
1

Hello Expo — Your First App on Your Phone

⏱ 45 min
Goal: Create, run, and edit an Expo app. See hot reload working on your physical device within 10 minutes of starting.

Step 1 — Install Node.js
Download and install Node.js LTS from nodejs.org. Verify: node --version should print v18 or higher.

node --version    # e.g. v20.11.0
npm --version     # e.g. 10.2.4

Step 2 — Create your Expo project

npx create-expo-app MyFirstApp --template blank
cd MyFirstApp
npx expo start
Tip: npx create-expo-app downloads and runs the latest Expo CLI without a global install. You do not need to run npm install -g expo-cli anymore.

Step 3 — Connect your phone
Install Expo Go from the App Store (iOS) or Play Store (Android). Open Expo Go, tap "Scan QR code", and scan the QR code shown in your terminal. Your app appears on your device.

Step 4 — Explore the folder structure

MyFirstApp/
├── App.js          # your root component — start here
├── app.json        # app name, bundle ID, icon
├── package.json    # dependencies and scripts
├── assets/         # icon.png, splash.png, adaptive-icon.png
└── node_modules/   # never edit these files

Step 5 — Edit App.js and see hot reload
Open App.js in VS Code. Change the text inside <Text> to your name. Save the file. Watch your phone update within 1–2 seconds.

import { StyleSheet, Text, View } from 'react-native';

Todo App — Core Components & useState

⏱ 60 min
Goal: Build a functional todo list using View, Text, TextInput, TouchableOpacity, and FlatList. Add and delete todos with useState.

Step 1 — Create a new project

npx create-expo-app TodoApp --template blank
cd TodoApp
npx expo start

Step 2 — Build the full App.js

import { useState } from 'react';
import { View, Text, TextInput, TouchableOpacity, FlatList, StyleSheet, Alert } from 'react-native';

export default function App() {
  const [todos, setTodos] = useState([]);
  const [input, setInput]  = useState('');

  function addTodo() {
    if (!input.trim()) { Alert.alert('Please type something first'); return; }
    setTodos([...todos, { id: Date.now().toString(), text: input.trim() }]);
    setInput('');
  }

  function deleteTodo(id) {
    setTodos(todos.filter(t => t.id !== id));
  }

  return (
    <View style={styles.screen}>
      {/* Header */}
      <View style={styles.header}>
        <Text style={styles.headerTitle}>My Todos</Text>
        <Text style={styles.headerCount}>{todos.length} items</Text>
      </View>

      {/* Input row */}
      <View style={styles.inputRow}>
        <TextInput
          style={styles.input}
          value={input}
          onChangeText={setInput}
          placeholder="Add a new todo..."
          onSubmitEditing={addTodo}
          returnKeyType="done"
        />
        <TouchableOpacity style={styles.addBtn} onPress={addTodo} activeOpacity={0.8}>
          <Text style={styles.addBtnText}>Add</Text>
        </TouchableOpacity>
      </View>

      {/* List */}
      <FlatList
        data={todos}
        keyExtractor={item => item.id}
        contentContainerStyle={todos.length === 0 ? styles.emptyContainer : {padding: 16}}
        ListEmptyComponent={
          <Text style={styles.emptyText}>No todos yet — add one above!</Text>
        }
        renderItem={({ item }) => (
          <View style={styles.todoCard}>
            <Text style={styles.todoText}>{item.text}</Text>
            <TouchableOpacity onPress={() => deleteTodo(item.id)} style={styles.deleteBtn}>
              <Text style={styles.deleteBtnText}>✕</Text>
            </TouchableOpacity>
          </View>
        )}
      />
    </View>
  );
}

const styles = StyleSheet.create({
  screen:         { flex: 1, backgroundColor: '#F8F6F0' },
  header:         { backgroundColor: '#1B3A6B', padding: 20, paddingTop: 60,
                    flexDirection: 'row', justifyContent: 'space-between', alignItems: 'flex-end' },
  headerTitle:    { fontSize: 24, fontWeight: '800', color: 'white' },
  headerCount:    { fontSize: 13, color: '#C8941A', fontWeight: '600' },
  inputRow:       { flexDirection: 'row', margin: 16, gap: 10 },
  input:          { flex: 1, backgroundColor: 'white', borderRadius: 10, padding: 14,
                    fontSize: 14, borderWidth: 1, borderColor: '#D8D4CC' },
  addBtn:         { backgroundColor: '#C8941A', borderRadius: 10, paddingHorizontal: 20,
                    justifyContent: 'center' },
  addBtnText:     { color: 'white', fontWeight: '700', fontSize: 14 },
  todoCard:       { backgroundColor: 'white', borderRadius: 12, padding: 16, marginBottom: 10,
                    flexDirection: 'row', alignItems: 'center',
                    shadowColor: '#000', shadowOffset: {width:0,height:1},

Checkpoint: Todos appear as cards, can be added and deleted, and the count in the header updates correctly.

Step 3 — Wire navigation in App.js

import { NavigationContainer } from '@react-navigation/native';
import { createNativeStackNavigator } from '@react-navigation/native-stack';
import { createBottomTabNavigator } from '@react-navigation/bottom-tabs';

const Stack = createNativeStackNavigator();
const Tab   = createBottomTabNavigator();

function HomeTabs() {
  return (
    <Tab.Navigator screenOptions={{ tabBarActiveTintColor: '#1B3A6B', headerShown: false }}>
      <Tab.Screen name="Trails" component={HomeScreen}
        options={{ tabBarIcon: () => <Text>🏔️</Text> }} />
      <Tab.Screen name="Profile" component={ProfileScreen}
        options={{ tabBarIcon: () => <Text>👤</Text> }} />
    </Tab.Navigator>
  );
}

export default function App() {
  return (
    <NavigationContainer>
      <Stack.Navigator>
        <Stack.Screen name="Main" component={HomeTabs} options={{ headerShown: false }} />
        <Stack.Screen name="Detail" component={DetailScreen}
          options={{ title: 'Trail Detail', headerBackTitle: 'Back' }} />
      </Stack.Navigator>
    </NavigationContainer>
  );
}

Lab 4 — Form with Validation (Week 4)

Registration Form — Controlled Inputs & Validation

⏱ 60 min
Goal: Build a registration form that validates all fields inline, disables Submit while invalid, and uses KeyboardAvoidingView so the keyboard never hides the inputs.
import { useState } from 'react';
import {
  View, Text, TextInput, TouchableOpacity,
  StyleSheet, KeyboardAvoidingView, ScrollView, Platform, Alert
} from 'react-native';

export default function RegisterScreen() {
  const [form, setForm] = useState({ name: '', email: '', password: '', confirm: '' });
  const [errors, setErrors] = useState({});

  function update(field, value) {
    setForm({ ...form, [field]: value });
    setErrors({ ...errors, [field]: '' }); // clear error on type
  }

  function validate() {
    const e = {};
    if (!form.name.trim())   e.name     = 'Name is required';
    if (!/\S+@\S+\.\S+/.test(form.email)) e.email = 'Enter a valid email address';
    if (form.password.length < 8) e.password = 'Password must be at least 8 characters';
    if (form.password !== form.confirm) e.confirm = 'Passwords do not match';
    setErrors(e);
    return Object.keys(e).length === 0;
  }

  function handleSubmit() {
    if (validate()) Alert.alert('Success', `Welcome, ${form.name}!`);
  }

  const isFormFilled = form.name && form.email && form.password && form.confirm;

  return (
    <KeyboardAvoidingView
      style={{ flex: 1 }}
      behavior={Platform.OS === 'ios' ? 'padding' : 'height'}
    >
      <ScrollView contentContainerStyle={styles.container} keyboardShouldPersistTaps="handled">
        <Text style={styles.title}>Create Account</Text>
        <Text style={styles.subtitle}>NextGen Devbhoomi Internship 2026–27</Text>

        {[
          { key: 'name',     label: 'Full Name',       placeholder: 'Priya Sharma' },
          { key: 'email',    label: 'Email Address',   placeholder: 'priya@example.com', keyboard: 'email-address', autoCapitalize: 'none' },
          { key: 'password', label: 'Password',        placeholder: 'Min 8 characters',  secure: true },
          { key: 'confirm',  label: 'Confirm Password', placeholder: 'Repeat password',   secure: true },
        ].map(f => (
          <View key={f.key} style={styles.fieldWrap}>
            <Text style={styles.label}>{f.label}</Text>
            <TextInput
              style={[styles.input, errors[f.key] && styles.inputError]}
              value={form[f.key]}
              onChangeText={v => update(f.key, v)}
              placeholder={f.placeholder}
              keyboardType={f.keyboard || 'default'}
              autoCapitalize={f.autoCapitalize || 'words'}
              secureTextEntry={!!f.secure}
              autoCorrect={false}
            />
            {errors[f.key] && <Text style={styles.errorText}>{errors[f.key]}</Text>}
          </View>
        ))}

        <TouchableOpacity
          style={[styles.submitBtn, !isFormFilled && styles.submitDisabled]}
          onPress={handleSubmit}
          disabled={!isFormFilled}
          activeOpacity={0.8}
        >
          <Text style={styles.submitText}>Create Account</Text>
        </TouchableOpacity>
      </ScrollView>
    </KeyboardAvoidingView>
  );
}

const styles = StyleSheet.create({
  container:      { flexGrow: 1, backgroundColor: '#F8F6F0', padding: 24, paddingTop: 60 },
  title:          { fontSize: 28, fontWeight: '800', color: '#1B3A6B', marginBottom: 4 },
  subtitle:       { fontSize: 13, color: '#5A6070', marginBottom: 32 },
  fieldWrap:      { marginBottom: 16 },
  label:          { fontSize: 13, fontWeight: '600', color: '#1B3A6B', marginBottom: 6 },
  input:          { backgroundColor: 'white', borderWidth: 1, borderColor: '#D8D4CC',
                    borderRadius: 10, padding: 14, fontSize: 14 },
  inputError:     { borderColor: '#C62828' },
  errorText:      { fontSize: 12, color: '#C62828', marginTop: 4 },
  submitBtn:      { backgroundColor: '#1B3A6B', borderRadius: 12, padding: 16,
                    alignItems: 'center', marginTop: 8 },
  submitDisabled: { opacity: 0.4 },
  submitText:     { color: 'white', fontWeight: '700', fontSize: 15 },
});
Checkpoint: Inline errors appear on submit, the button stays greyed-out until all fields have content, and the keyboard never covers the active input.
Lab 5 — Local Storage Notes App (Week 4)

Notes App — AsyncStorage Persistence

⏱ 75 min
Goal: Build a notes app that saves data to AsyncStorage. Notes survive closing and reopening the app. Shows empty state when no notes exist.

Step 1 — Install AsyncStorage

npx expo install @react-native-async-storage/async-storage

Step 2 — Build the Notes App

import { useState, useEffect } from 'react';
import {
  View, Text, TextInput, TouchableOpacity,
  FlatList, StyleSheet, Alert, Modal
} from 'react-native';
import AsyncStorage from '@react-native-async-storage/async-storage';

const STORAGE_KEY = '@notes_v1';

export default function App() {
  const [notes, setNotes] = useState([]);
  const [modalVisible, setModalVisible] = useState(false);
  const [editNote, setEditNote] = useState(null);
  const [title, setTitle]   = useState('');
  const [body, setBody]     = useState('');

  // Load notes on mount
  useEffect(() => {
    (async () => {
      const raw = await AsyncStorage.getItem(STORAGE_KEY);
      if (raw) setNotes(JSON.parse(raw));
    })();
  }, []);

  // Save whenever notes change
  useEffect(() => {
    AsyncStorage.setItem(STORAGE_KEY, JSON.stringify(notes));
  }, [notes]);

  function openAdd() {
    setEditNote(null); setTitle(''); setBody(''); setModalVisible(true);
  }

  function openEdit(note) {
    setEditNote(note); setTitle(note.title); setBody(note.body); setModalVisible(true);
  }

  function saveNote() {
    if (!title.trim()) { Alert.alert('Add a title'); return; }
    if (editNote) {
      setNotes(notes.map(n => n.id === editNote.id ? { ...n, title, body } : n));
    } else {
      setNotes([{ id: Date.now().toString(), title: title.trim(), body: body.trim() }, ...notes]);
    }
    setModalVisible(false);
  }

  function deleteNote(id) {
    Alert.alert('Delete note?', 'This cannot be undone.', [
      { text: 'Cancel' },
      { text: 'Delete', style: 'destructive', onPress: () => setNotes(notes.filter(n => n.id !== id)) },
    ]);
  }

  return (
    <View style={styles.screen}>
      <View style={styles.header}>
        <Text style={styles.headerTitle}>Notes</Text>
        <TouchableOpacity style={styles.addBtn} onPress={openAdd}>
          <Text style={styles.addBtnText}>+ New</Text>
        </TouchableOpacity>
      </View>

      <FlatList
        data={notes}
        keyExtractor={n => n.id}
        contentContainerStyle={notes.length === 0 ? styles.emptyWrap : { padding: 16 }}
        ListEmptyComponent={<Text style={styles.emptyText}>No notes yet.{'\n'}Tap + New to create one.</Text>}
        renderItem={({ item }) => (
          <TouchableOpacity style={styles.noteCard} onPress={() => openEdit(item)} activeOpacity={0.85}>
            <Text style={styles.noteTitle} numberOfLines={1}>{item.title}</Text>
            <Text style={styles.noteBody} numberOfLines={2}>{item.body || 'No body text'}</Text>
            <TouchableOpacity onPress={() => deleteNote(item.id)} style={styles.delBtn}>
              <Text style={styles.delBtnText}>Delete</Text>
            </TouchableOpacity>
          </TouchableOpacity>
        )}
      />

      {/* Add/Edit Modal */}
      <Modal visible={modalVisible} animationType="slide" presentationStyle="pageSheet">
        <View style={styles.modal}>
          <View style={styles.modalHeader}>
            <TouchableOpacity onPress={() => setModalVisible(false)}>
              <Text style={styles.cancel}>Cancel</Text>
            </TouchableOpacity>
            <Text style={styles.modalTitle}>{editNote ? 'Edit Note' : 'New Note'}</Text>
            <TouchableOpacity onPress={saveNote}>
              <Text style={styles.save}>Save</Text>
            </TouchableOpacity>
          </View>
          <TextInput style={styles.titleInput} value={title} onChangeText={setTitle} placeholder="Title" />
          <TextInput style={styles.bodyInput} value={body} onChangeText={setBody}
            placeholder="Write your note here..." multiline />
        </View>
      </Modal>
    </View>
  );
}

const styles = StyleSheet.create({
  screen:      { flex: 1, backgroundColor: '#F8F6F0' },
  header:      { backgroundColor: '#1B3A6B', paddingTop: 56, paddingBottom: 16, paddingHorizontal: 20,
                 flexDirection: 'row', justifyContent: 'space-between', alignItems: 'center' },
  headerTitle: { fontSize: 26, fontWeight: '800', color: 'white' },
  addBtn:      { backgroundColor: '#C8941A', paddingHorizontal: 16, paddingVertical: 8,
                 borderRadius: 8 },
  addBtnText:  { color: 'white', fontWeight: '700', fontSize: 13 },
  noteCard:    { backgroundColor: 'white', borderRadius: 12, padding: 16, marginBottom: 12,
                 shadowColor: '#1B3A6B', shadowOffset: {width:0,height:2},
                 shadowOpacity: 0.07, shadowRadius: 6, elevation: 2 },
  noteTitle:   { fontSize: 15, fontWeight: '700', color: '#1B3A6B', marginBottom: 4 },
  noteBody:    { fontSize: 13, color: '#5A6070', lineHeight: 18, marginBottom: 10 },
  delBtn:      { alignSelf: 'flex-end' },
  delBtnText:  { fontSize: 11, color: '#C62828', fontWeight: '600' },
  emptyWrap:   { flex: 1, justifyContent: 'center', alignItems: 'center' },
  emptyText:   { fontSize: 15, color: '#9AA0AE', textAlign: 'center', lineHeight: 24 },
  modal:       { flex: 1, backgroundColor: '#F8F6F0', padding: 20 },
  modalHeader: { flexDirection: 'row', justifyContent: 'space-between', alignItems: 'center',
                 marginBottom: 20 },
  modalTitle:  { fontSize: 16, fontWeight: '700', color: '#1B3A6B' },
  cancel:      { fontSize: 14, color: '#5A6070' },
  save:        { fontSize: 14, fontWeight: '700', color: '#1B3A6B' },
  titleInput:  { backgroundColor: 'white', borderRadius: 10, padding: 14, fontSize: 17,
                 fontWeight: '600', marginBottom: 12, borderWidth: 1, borderColor: '#D8D4CC' },
  bodyInput:   { backgroundColor: 'white', borderRadius: 10, padding: 14, fontSize: 14,
                 borderWidth: 1, borderColor: '#D8D4CC', minHeight: 200, textAlignVertical: 'top' },
});
Checkpoint: Close and reopen the app — all notes are still there. Add, edit, and delete all work correctly. Empty state shows when no notes exist.
Tip: To verify persistence, close the app completely (swipe it away from the app switcher), then reopen. If notes still appear, AsyncStorage is working correctly.
