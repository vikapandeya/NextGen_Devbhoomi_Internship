Job Roles in DevOps & Cloud
🌱

Junior DevOps Engineer

0–1 Year Experience
₹4–7 LPA
  • Linux basics and terminal comfort
  • Git and basic branching
  • Docker — build and run containers
  • Basic CI/CD pipeline understanding
  • Basic AWS console navigation
⚡

DevOps Engineer

1–3 Years Experience
₹7–15 LPA
  • Docker + Kubernetes in production
  • GitHub Actions / Jenkins pipelines
  • Terraform for infrastructure
  • AWS or Azure at scale
  • Monitoring with Prometheus/Grafana
☁️

Cloud Engineer

2–4 Years Experience
₹8–18 LPA
  • Multi-cloud architecture (AWS + Azure)
  • IaC at scale with Terraform modules
  • VPC design and network security
  • Cost optimisation and FinOps
  • Cloud security and compliance
🔧

Site Reliability Engineer (SRE)

3–5 Years Experience
₹15–30 LPA
  • SLO/SLA/SLI definition and tracking
  • Incident response and postmortems
  • Chaos engineering (Gremlin, LitmusChaos)
  • Platform engineering
  • Toil reduction and automation
🏗️

Cloud Architect

6+ Years Experience
₹30–60+ LPA
  • Multi-region disaster recovery design
  • Organisation-wide cloud standards
  • Well-Architected Framework reviews
  • Migration strategy for legacy systems
  • Executive-level technical communication
💼

DevOps / Platform Lead

5+ Years Experience
₹25–55+ LPA
  • Team leadership and hiring
  • Internal Developer Platform (IDP) design
  • FinOps — cloud spend accountability
  • Define team DevOps practices
  • Cross-team incident coordination
Certifications That Open Doors
☁️

AWS Cloud Practitioner

Foundation cloud concepts, billing, and security. The best first certification — validates cloud literacy to any employer. ~₹9,000 exam fee.

Foundation AWS
🏆

AWS Solutions Architect Associate

Design fault-tolerant, scalable architectures. The most valued AWS certification for DevOps/Cloud roles in India. ~₹15,000 exam fee.

Associate AWS
⎈

Certified Kubernetes Administrator (CKA)

Hands-on K8s cluster administration — all practical exam, no multiple choice. Growing demand for this cert in product companies. ~₹35,000 exam fee.

Advanced CNCF
🌐

HashiCorp Terraform Associate

IaC with Terraform — validates you can write, plan, and apply infrastructure as code. Demand growing as companies adopt IaC. ~₹12,000 exam fee.

Associate HashiCorp
🔵

Google Cloud Associate Cloud Engineer

GCP equivalent of the AWS SAA. Valuable if targeting Google Cloud India, Deloitte, or companies migrating to GCP. ~₹15,000 exam fee.

Associate GCP
🪟

Azure AZ-104 Administrator

Azure administration — particularly valuable at TCS, Infosys, Wipro, and HCL where Microsoft Azure is the dominant cloud. ~₹12,000 exam fee.

Associate Microsoft
Skill Milestones by Week

🌱 Foundational (Weeks 1–4)

  • Use Linux terminal comfortably
  • Git feature branch workflow
  • Basic AWS console navigation
  • Explain IaaS / PaaS / SaaS
  • Set up billing alarm on AWS

⚡ Application (Weeks 5–8)

  • Dockerize any application
  • Build a working CI/CD pipeline
  • Deploy to EC2 with security groups
  • Write basic Terraform code
  • Push images to AWS ECR

🚀 Professional (Weeks 9–12)

  • K8s deployment with auto-healing
  • Monitoring with Prometheus alerts
  • Security scanning in CI pipeline
  • Define SLOs for a service
  • HPA under simulated load

💼 Job-Ready Signal

  • 2+ deployed projects with CI/CD
  • AWS cert or actively in progress
  • Active GitHub with green squares
  • Resume highlights cost savings or uptime
  • Can draw CI/CD pipeline from memory
90-Day Action Plan After Internship

🎯 Your 90-Day Launch Plan

Days 1–30: Get Certified, Publish Work

  • Take the AWS Cloud Practitioner exam — book it now
  • Deploy all 3 projects with live URLs and CI/CD status badges on GitHub README
  • Write a LinkedIn post about your DevOps journey — tag #DevOps #AWS
  • Set up a GitHub Pages blog or writeup for each project — "What I built and why"
  • Connect with 20 DevOps engineers on LinkedIn with a real message, not a template

Days 31–60: Apply Strategically

  • Apply to DevOps / Cloud Engineer roles on LinkedIn, Naukri, and Indeed
  • Target IT services digital teams (TCS, Infosys Cobalt) AND product companies
  • Practice explaining: CI/CD pipeline, Docker vs VM, Terraform state, K8s architecture
  • Contribute one small fix to an open-source DevOps tool on GitHub
  • Ask one mentor or senior engineer to review your resume

Days 61–90: Interview and Land the Offer

  • Know your projects cold — every architecture decision and why you made it
  • Practice whiteboarding: draw a CI/CD pipeline from scratch in 5 minutes
  • Know your numbers: which AWS services cost what, what your projects ran at
  • Negotiate — DevOps skills are genuinely scarce. You have more leverage than you think
  • Accept the offer that gives you the most learning, not just the highest salary
